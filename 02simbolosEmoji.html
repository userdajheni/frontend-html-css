Emojis

Outra coisa que você pode adicionar às suas páginas são os emojis, onde cada símbolo possui um código Unicode. Para uma pesquisa mais precisa e atualizada, acesse o site: Emojipedia. Antes do código informado insira &#x e depois o código 1F6F8 (cada emoji possui o seu).

Se utilizar os emojis do W3School o código inicia-se apenas com &#. Verifique o melhor para você.
