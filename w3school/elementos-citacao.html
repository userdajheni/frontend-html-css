Elementos de Citação

Nesta seção veremos: blockquote, q, abbr, address, cite e bdo



Blockquote

Este elemento define uma seção que é citado de outra fonte. Os navegadores geralmente indentam o conteudo de blockquote. 

           < blockquote cite="http://www.worldwildlife.org/who/index.html">
                For 60 years, WWF has worked to help people and nature thrive. As the world's leading conservation organization, WWF works in nearly 100 countries. 
                At every level, we collaborate with people around the world to develop and deliver innovative solutions that protect communities, wildlife, and the places in which they live.
            < /blockquote >



Usamos q para citações curtas

Os navegadores normalmente inserem aspas ao redor da citação. 

            < p >WWF's goal is to: Estou dentro de uma tag q < /p >



Usamos abbr para abreviaturas

Esta tag define uma abreviação ou um acrônimo, como "HTML", "CSS", "Sr.", "Dr.", "ASAP", "ATM". Marcar abreviaturas pode fornecer informações úteis para navegadores, tradução sistemas e motores de busca. Use o atributo title para mostrar a descrição para a abreviação/acrônimo quando você passar o mouse sobre o elemento. 

            < p >The passe o mouse sobre mim was founded in 1948.< /p >



Usamos address para informações de contato

Esta tag define as informações de contato do autor/proprietário de um documento ou um artigo. As informações de contato podem ser um endereço de e-mail, URL, endereço físico, telefone número, identificador de mídia social, etc. O texto no elemento < address > geralmente renderiza em itálico, e os navegadores irão sempre adicionar uma quebra de linha antes e depois deste elemento. 

            Lorem ipsum dolor sit amet consectetur, adipisicing elit. Minima, optio?
            

                Written by John Doe.

                Visit us at:

                Example.com

                Box 564, Disneyland

                USA
            
 
            Lorem ipsum dolor sit, amet consectetur adipisicing elit. Ipsum quibusdam id sint. Enim, rerum dignissimos.



Usamos cite para o título do trabalho

Esta tag define o título de um trabalho criativo (por exemplo, um livro, um poema, uma música, um filme, uma pintura, uma escultura, etc.). 

            
The Scream by Edvard Munch. Painted in 1893.



Usamos bdo para para substituição bidirecional

Esta tag é usada para substituir a direção do texto atual

        texto dentro de um bdo com direção da Direita para a Esquerda
        texto dentro de um bdo com direção da Esquerda para a Direita
