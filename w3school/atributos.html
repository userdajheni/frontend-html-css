Atributos

Todos os elementos HTML podem ter atributos. Eles fornecem informações adicionais sobre os elementos e são sempre especificados na tag de início, geralmente vêm em pares de nome/valor como: name="value".

  • href --> especifica a URL de uma página
  • src --> é usada para incorporar um imagem
  • width --> largura
  • height --> altura
  • Alt --> Usado na tag img para mostrar um texto caso a imagem não seja carregada
  • style --> usado para adicionar estilos a um elemento
  • title --> O valor do atributo title será exibido como uma dica de ferramenta quando você passa o mouse sobre o elemento
