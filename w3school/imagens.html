Imagens

As imagens podem melhorar o design e a aparência de uma página da web. A tag img é usada para incorporar uma imagem. O atributo src especifica o caminho para a imagem e o atributo alt especifica um texto alternativo para a imagem. 

            < img src="img_chania.jpg" alt="Flowers in Chania" >



Image Maps

Com mapas de imagem você pode criar áreas clicáveis ​​em uma imagem.

A tag map define um mapa de imagem. Um mapa de imagem é uma imagem com áreas clicáveis. As áreas são definidas com um ou mais tags area. Na imagem abaixo clique no computador, depois no celular e depois na xícara de café:

A ideia por trás de um mapa de imagem é que você deve ser capaz de realizar diferentes ações dependendo de onde você clicar na imagem. Uma área clicavél é definida pela tag area e você pode escolher as seguintes formas:

  • rect- define uma região retangular
  • circle- define uma região circular
  • poly- define uma região poligonal
  • default- define toda a região
