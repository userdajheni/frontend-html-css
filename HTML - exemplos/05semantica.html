Cabeçalho

O 'header' A tag é usada para marcar a seção superior de uma página da Web, que normalmente inclui o título ou logotipo principal, links de navegação e outros elementos comuns a todas as páginas do site. Com ele você pode criar uma separação clara entre o cabeçalho e o conteúdo principal da sua página, tornando mais fácil para os usuários entender e navegar no seu site.

Navegação

A tag 'nav' é usada para marcar uma seção de uma página da web que contém links de navegação. O nav é o menu.

Main

A tag 'main' é usada para marcar a área de conteúdo principal de uma página da web, indica qual parte da sua página contém o conteúdo principal.

Article

A tag 'article' é usada para representar um conteúdo independente, como uma postagem de blog, artigo de notícias ou revisão de produto. O conteúdo desta tag deve ser independente e significativa e pode incluir cabeçalhos, parágrafos, imagens e outros tipos de conteúdo.

Aside

A tag 'aside' é usada para marcar o conteúdo relacionado ao conteúdo principal de uma página da Web, mas não é parte integrante dela. Isso pode incluir coisas como informações complementares, anúncios ou artigos relacionados.

Section

A tag 'section' é usada para marcar seções de uma página da Web agrupadas tematicamente. Isso pode incluir itens como capítulos ou seções de um artigo longo ou diferentes partes de uma página de produto.

Footer

É a seção inferior de uma página da web, que pode incluir informações como informações de direitos autorais, detalhes de contato ou links para perfis de mídia social.

Details e summary

Os dois são usados para marcar uma seção recolhível do conteúdo: summary marca o título e details marca o próprio conteúdo, ou seja, summary está dentro de details.

Figure e Figcaption

São usados para marcar um conteúdo independente que é referenciado no conteúdo principal de uma página da web. O figure marca o conteúdo e figcaption marca a legenda, ou seja, figcaption está dentro de figure.
