Parágrafos

A tag < p > indica o elemento parágrafo. Possui uma quebra de linha própria.

Isto é um parágrafo.

Citações

blockquote

Define uma citação longa, exemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Explicabo tempora eveniet reprehenderit illum non mollitia rem maiores nam ducimus deserunt!



q

Define uma citação curta, exemplo:

Lorem ipsum, dolor sit amet consectetur adipisicing elit. Modi, eaque.


abbr

Define uma abreviação e geralmente inclui sua descrição completa, exemplo:

O HTML é a estrutura de um site.




address

Define um bloco para informações de contato, exemplo:

Infinite Loop,
Cupertino, CA
95014, USA



cite

Define a fonte de um trabalho criativo, exemplo:

Se você quiser aprender HTML e CSS, leia MarkSheet.io!




bdo

Permite substituir a direção do texto.

Texto normal: Olá

Texto invertido: Olá

Texto invertido novamente: Olá
