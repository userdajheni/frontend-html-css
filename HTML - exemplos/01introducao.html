O que é html?

HTML significa HyperText Markup Language. Ele é usado no front-end e fornece a estrutura da página da Web que você pode estilizar usando CSS e tornar interativa usando JavaScript. É uma linguagem de marcação padrão para criar páginas da Web. Consiste em uma série de elementos que informam ao navegador como exibir o conteúdo, esses elementos rotulam partes do conteúdo como "este é um cabeçalho", "este é um parágrafo", "este é um link", etc.

HTML é a linguagem na qual a maioria dos sites são escritos . O HTML é usado para criar páginas e torná-las funcionais. O código usado para torná-los visualmente atraentes é conhecido como CSS e o traz interatividade é o javascript. HTML é apenas a estrutura da página, o alicerce.



Exemplo explicado

O código acima é a estrutura básica necessária para um arquivo html funcionar. Veja o que significa cada elemento:

  • A declaração < !DOCTYPE html > define que este documento é um documento HTML5

  • O elemento < html > é a raiz de uma página html

  • O elemento < head > contém metainformações sobre a página HTML. O conteúdo do header não aparece para o usuário

  • O elemento < title > especifica um título para o Página HTML (que é mostrada na aba do navegador)

  • O elemento < body > define o corpo do documento e é um contêiner para todo o conteúdo visível, como títulos, parágrafos, imagens, hiperlinks, tabelas, listas, etc. Tudo o que estiver dentro de body aparecerá para o usuário

  • O elemento < h1 > define um título de nível 1. Existem 6 níveis.

  • O elemento < p > define um parágrafo
