Atributos

Um atributo fornece informações adicionais sobre os elementos HTML. Todos os elementos HTML podem ter atributos, são sempre especificados na tag de início e geralmente vêm em pares de nome/valor como: name="value". O href é utilizado pelas tags a e link.

Atributo href

O atributo href especifica a URL da página, para onde o link será direcionado, faz referência a uma URL externa, seja um link ou um arquivo. Exemplos:

Href sendo usado em uma tag a para criar um link

< a href="https://www.youtube.com.br"> Acesse o youtube < /a>

CSS sendo carregado através da tag link

< link rel="stylesheet" type="text/css" href="../css/estilo.css" />


Atributo src

Este atributo especifica o nome do caminho de um arquivo de mídia. O arquivo pode ser qualquer tipo: imagem, arquivo javascript, video, etc.

Chamando uma imagem: < img src="../imagens/nome-da-imagem.png" >

Carregando um arquivo externo: < script src="../pasta/nome-do-arquivo.js">< /script >




Atributo Alt

Aplica-se na tag img e especifica um texto alternativo para uma imagem, se a imagem por algum motivo não puder ser exibida.

< img src="../imagens/nome-da-imagem.png" alt="logo html">
