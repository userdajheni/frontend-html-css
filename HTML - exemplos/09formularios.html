Formulários

Um formulário HTML é usado para coletar a entrada do usuário. A entrada do usuário é na maioria das vezes enviados para um servidor para processamento. O elemento <'form'> é usado para crie um formulário HTML para entrada do usuário. O elemento <'label'> define um rótulo, ele é útil para usuários com leitores de tela, pois o leitor de tela lerá em voz alta o rótulo quando o usuário se concentra no elemento de entrada. Atributos

O <'action'> define a ação a ser executada quando o formulário é enviado. Normalmente, os dados do formulário são enviados para um arquivo no servidor quando o usuário clica no botão enviar. O atributo target especifica onde exibir a resposta recebida após o envio do formulário. Pode ter os seguintes valores:

  • _blank - A resposta é exibida em uma nova janela ou guia
  • _self - A resposta é exibida na janela atual
  • _parent - A resposta é exibida no quadro pai
  • _top - A resposta é exibida no corpo inteiro da janela
  • framename - A resposta é exibida em um iframe nomeado



Método GET e POST

O atributo method especifica qual método http será usado ao enviar os dados do formulário, sendo GET ou POST.

O método GET anexa os dados do formulário ao URL, em pares de nome/valor, os dados do formulário enviado são visíveis na URL por este motivo este método não é seguro. Ele é útil para envios de formulários em que um usuário deseja marcar o resultado como favorito e também é bom para dados não seguros, como strings de consulta no Google.

O método POST anexa os dados do formulário dentro do corpo da solicitação HTTP (o enviado os dados do formulário não são mostrados na URL). É um método mais seguro que o GET. Este método não tem limitações de tamanho e pode ser usado para enviar grandes quantidades de dados.



Preenchimento automático

O atributo autocomplete especifica se um formulário deve ter o preenchimento automático ativado ou desativado. Quando o preenchimento automático está ativado, o navegador preenche automaticamente os valores com base nos valores que o usuário inseriu anteriormente.



Atributo Novalidate

O atributo novalidate é um atributo booleano. Quando presente, especifica que os dados do formulário (input) não devem ser validados quando enviados.
