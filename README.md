# HTML
HTML significa HyperText Markup Language. Ele é usado no front-end e fornece a estrutura da página da Web que você pode estilizar usando CSS e tornar interativa usando JavaScript. O HTML é uma linguagem de marcação que define a estrutura do seu conteúdo. Consiste em uma série de elementos que você usa para mostrar algo de uma determinada maneira ou agir de uma certo modo. As tags podem criar um hiperlink de uma palavra ou imagem para outro lugar, podem colocar palavras em itálico, podem aumentar ou diminuir a fonte e assim por diante.
