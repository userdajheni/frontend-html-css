CSS Selectors

CSS são a combinação de caracteres e símbolos especiais que instrui o navegador a aplicar propriedades CSS específicas ao elemento HTML desejado. Os seletores CSS usam o nome id (começa com #), nome da classe, tipo da tag HTML (como cabeçalho, div, etc.), atributo, etc. do elemento HTML para selecioná-los. O(s) elemento(s) HTML que é(são) selecionado(s) pelo seletor é(são) chamado(s) de assunto do seletor.

Seletores CSS Básicos

  • Seletor Universal:
    Ele seleciona todos os elementos HTML presentes na página HTML. Para usar o seletor universal, precisamos usar asteriscos (*) e depois definir a propriedade CSS e depois disso, todos os elementos serão selecionados.

  • Seletores de tipo:
    Um seletor de tipo também é conhecido como seletor de elemento porque seleciona tag(s)/elemento(s) HTML em seu documento. O seletor de elementos seleciona elementos HTML com base em seus nomes, alguns deles são p (tag de parágrafo), h1 (título 1), div (tag de divisão), span, etc. Para usar um seletor de tipo, você só precisa dar o nome da tag HTML e, em seguida, definir as propriedades CSS entre chaves.

  • Seletor de classe: O seletor de classe em CSS seleciona elementos que pertencem a um determinado atributo de classe. Para selecionar elementos com uma classe específica, escreva um ponto ou ponto final (.) e, em seguida, o nome da classe.

  • Seletor de ID: O seletor de ID de CSS usa o atributo de ID do elemento HTML para selecionar elementos em uma página e aplicar propriedades CSS a eles. O atributo id é o identificador único em um documento HTML Para usar um seletor de ID em CSS, use uma hashtag (#) seguida do nome do ID do elemento HTML. O elemento HTML só será selecionado se seu atributo id corresponder exatamente ao valor informado no seletor.


Seletores de Atributo

Seletor [Attribute]: Nesse tipo de seletor de atributo, o nome do elemento é usado e, em seguida, o nome do atributo é especificado entre colchetes. Os elementos que correspondem ao nome do atributo especificado no colchete são selecionados e as propriedades CSS são aplicadas a eles.



Se você deseja aplicar uma propriedade CSS específica a diferentes elementos que contêm o mesmo atributo, nesse caso, especifique os diferentes elementos e separe-os por vírgula (,).



Seletor [atributo = “valor”]: Neste tipo de seletor de atributo, todos os elementos HTML cujo atributo e valor do atributo (Valor do atributo é a string dos caracteres que são atribuídos ao atributo do elemento HTML. Mencionado entre aspas duplas “ ”) corresponde exatamente ao os valores especificados no seletor são selecionados e as propriedades CSS especificadas são aplicadas.



Seletor [attr~=value]: Usando este tipo de seleção de atributo também, selecionamos elementos HTML com base no atributo e seus valores. Nesse caso, o elemento será selecionado mesmo que o valor mencionado no seletor corresponda a qualquer valor (já que eles podem ser uma lista de palavras) no atributo do elemento.



Seletor [atributo|=valor]: Este seletor de atributo é usado para selecionar todos os elementos HTML cujo atributo tem um valor de uma lista de palavras separadas por hífen, o elemento será selecionado se o início de uma lista de palavras separadas por hífen nos valores de atributo corresponder ao valor especificado em o seletor. O valor do atributo pode ser uma palavra inteira exata sozinha ou deve começar com o valor imediatamente seguido por um hífen. (-)



Seletor [attributo ^= "valor"]: Neste seletor o valor não precisa ser separado por hífen ou espaço, o valor do atributo só precisa começar com o valor especificado no seletor. No exemplo abaixo o estilo será aplicado apenas nas classes que começam com colored.



Seletor [attribute$="value"]: Neste seletor selecionamos elementor que contenham atributos com valor final especificado. No exemplo abaixo receberá estilo apenas os elementos que possuem atributos que possuem algum valor com final "end".



Seletor [atributo *= "valor"]: Neste tipo de seletor de atributo, podemos selecionar elementos cujo valor do atributo coincida com o valor especificado no seletor e o valor pode estar presente em qualquer lugar (se houver uma lista de palavras no valor). Seletor descendente (espaço): seleciona todos os elementos HTML que são descendentes de um elemento ancestral especificado. No exemplo a seguir, todo elemento p que estiver sob um elemento div receberá estilização.



Pseudo Classes

1 - Hover

Esta pseudo-classe adiciona um efeito especial ao elemento selecionado quando você aponta o cursor do mouse sobre ele, o que pode tornar o elemento interativo.



2 - first-child

O elemento especificado sempre será selecionado somente quando for o primeiro filho de qualquer elmento.
