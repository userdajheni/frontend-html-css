CSS Font Family

As fontes desempenham um papel vital em todos os sites. Eles não apenas tornam o texto legível e atraente, mas também prendem a atenção do usuário. É importante usar uma fonte que o usuário possa compreender. Além disso, usar o tamanho, esquema de cores, negrito/itálico, variante e peso corretos o torna interativo. A família de fontes em CSS é um conjunto de fontes que possuem designs e propriedades comuns. As fontes definidas na família de fontes diferem no peso da fonte, como negrito, itálico, etc. A propriedade font-family é usada em CSS para especificar a fonte de um texto. É uma lista priorizada de várias fontes, da prioridade mais alta à mais baixa, separada por vírgulas.



Tipos de família de fontes

Família de fontes específica: são usados ​​apenas para um tipo específico de fonte. Estes incluem Arial, Times New Roman, Tahoma , etc.
Família de fontes genéricas: são usadas para uma fonte geral e quase todos os navegadores as suportam.

Existem cinco famílias de fontes genéricas em CSS:

  • Serif inclui fontes com pontas ou traços pequenos e alados no final de cada letra.

  • Sans-serif inclui fontes muito mais limpas e sem aqueles traços no final de cada letra.

  • Monoespaçado Possuem letras da mesma largura. Dá uma aparência mecânica.

  • Cursive Eles são usados ​​para unir as letras usando traços conectivos e dar-lhes uma aparência manuscrita.

  • Fantasia São fontes decorativas que mantêm as características dos glifos do alfabeto tradicional não cursivo.



Propriedades da Fonte CSS

As diferentes propriedades de fonte CSS são:

  • Font-color: é usada para definir a cor do texto.

  • Font-size: é usada para definir ou ajustar o tamanho da fonte.

  • font-weight: O peso da fonte refere-se à espessura da fonte, ou seja, a espessura ou espessura da fonte na página da web. A propriedade font-weight do CSS é usada para definir o peso/espessura/espessura da fonte.

  • Font-Style: é usada para definir o estilo da fonte para o conteúdo de texto de um elemento. Aqui, estilo refere-se à variação no tipo de letra. Pode ser itálico, oblíquo ou normal (padrão) . A propriedade de estilo de fonte pode ser usada para decorar e atribuir importância a um texto específico.

  • Font-Variant: permite que você transforme a linha específica de texto ou parágrafo em versalete (letras maiúsculas que serão pequenas em tamanho em comparação com as letras maiúsculas normais). Ele também pode ter valores como normal (por padrão), inicial - Isso define a propriedade para o valor padrão e herdar - Leva o valor da propriedade do elemento pai para o elemento filho.

  • Font-Family: é um conjunto de fontes com tipos de letra semelhantes e designs comuns.

  • Line-Height: A altura da linha pode ser considerada a distância entre as linhas de texto em uma página da web. A propriedade line-height em CSS nos permite definir a altura da caixa de linha, ou seja, a quantidade de espaço acima e abaixo dos elementos inline.

  • Font-Stretch: é usada para esticar ou comprimir o texto horizontalmente e é usada para tornar o texto mais largo ou mais estreito.
