Cor CSS

Existem várias propriedades em CSS para definir e alterar a cor dos elementos. Essas propriedades possuem vários valores para definir a cor por meio do CSS. Você pode especificar os valores de cor em vários formatos, alguns dos exemplos dos formatos disponíveis são dados abaixo:

Formato Sintaxe Exemplo
Palavra-chave red, green, etc h1 { color: red; }
RGB % rgb(rrr%, ggg%, bbb%) h1 { cor: rgb(30%, 30%, 30%); }
RGB Absoluto rgb(rrr,ggg,bbb) h1 { cor: rgb(0, 255, 255); }
código hexadecimal #RRGGBB h1 { cor: #0000FF; }
Código Hexadecimal Curto #RGB h1 { cor: #00F; }
HSL hsl(valor de matiz, % de saturação, luminosidade%) h1 { cor: hsl(120, 50%, 25%); }
