Sintaxe CSS Básica

CSS é uma linguagem de folha de estilo usada para alterar a aparência de um documento produzido em uma linguagem de marcação, como HTML, alterando aspectos como cores, posicionamento e decorações. A seguir estão os componentes básicos de construção da sintaxe CSS:

  • Seletor: é a parte inicial de uma regra de sintaxe CSS. É uma coleção de elementos HTML que informa ao navegador quais elementos HTML devem receber os de propriedade CSS aplicados a eles.

  • Bloco de Declaração: é um bloco de declaração contém uma lista de declarações que definem como os elementos selecionados devem ser estilizados. Uma declaração consiste em uma propriedade e um valor, separados por dois pontos e finalizados por um ponto e vírgula.

  • Propriedade: é um identificador que especifica qual recurso, como cor ou tamanho da fonte, é levado em consideração.

  • Valor:os valores de CSS são alocados para Propriedades de CSS e salvos no de declaração de CSS , que faz parte da regra/declaração de . Por exemplo, a cor é definida como preto.


Sintaxe das fontes

A propriedade font em CSS é usada para customizar a aparência do texto. Você pode modificar o tamanho do texto, cor, estilo e muito mais usando a propriedade de fonte CSS. Estas são algumas propriedades importantes da fonte:

  • Font-color: atualiza a cor da fonte.

  • Font-family: Esta propriedade é usada para alterar o tipo da fonte, como Arial, sans-serif, etc.

  • Font-size: Atualiza o tamanho da fonte.

  • Estilo da fonte: A fonte pode ser em negrito, itálico ou oblíqua com este atributo.

  • Font-variant: Você pode usar a propriedade font-variant para converter o texto selecionado em letras maiúsculas.

  • Peso da fonte: permite alterar o negrito e a leveza da fonte.

Os font-size e font-family são obrigatórios. Se um dos outros valores estiver ausente, seu valor padrão será usado.
